## 🎨 **Customization Examples**

### **Change Dashboard Title**
Edit `microservices/ui/config/dashboard_config.json`:
```json
{
  "dashboard": {
    "title": {
      "en": "Your Custom Title",
      "ar": "عنوان مخصص"
    }
  }
}
```

### **Add New Metric**
```json
{
  "dashboard": {
    "metrics": {
      "custom_metric": {
        "en": "Custom Metric",
        "ar": "مقياس مخصص",
        "icon": "fas fa-star",
        "color": "primary"
      }
    }
  }
}
```

### **Change Theme Colors**
```json
{
  "theme": {
    "primary_color": "#your-color",
    "accent_color": "#your-accent"
  }
}
```
